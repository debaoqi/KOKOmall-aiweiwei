# 快速开始指南

## 三步完成图片上传

### 步骤 1: 查看需要的图片文件名

打开 `图片文件名列表.txt` 文件，你会看到所有需要上传的图片文件名。

每个文件名对应一个产品，格式为：`品牌_产品名称.jpg`

**示例：**
```
文件名: apple_iphone_13.jpg
产品: Apple iPhone 13 8+512G
价格: 4780 积分
```

### 步骤 2: 上传图片文件

将你的图片文件放入 `product-images-home` 文件夹中。

**重要提示：**
- 文件名必须与列表中的文件名**完全一致**
- 支持 `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif` 格式
- 建议使用 `.jpg` 格式以获得最佳兼容性

### 步骤 3: 运行自动更新脚本

在终端中运行：

```bash
cd product-images-home
python3 auto-update-html-images.py
```

脚本会自动：
- ✅ 扫描所有图片文件
- ✅ 更新HTML文件中的图片路径
- ✅ 创建备份文件（以防万一）

### 完成！

刷新浏览器，你的图片就会自动显示在首页的产品卡片中了！

## 常见问题

**Q: 图片上传后没有显示？**
A: 请检查：
1. 文件名是否正确（区分大小写）
2. 文件是否在 `product-images-home` 文件夹中
3. 是否运行了自动更新脚本

**Q: 可以批量上传吗？**
A: 可以！一次性将所有图片放入文件夹，然后运行一次更新脚本即可。

**Q: 图片格式有要求吗？**
A: 建议使用 `.jpg` 格式，也支持 `.png`, `.webp`, `.gif`。

**Q: 如果某个产品没有图片会怎样？**
A: 会自动显示默认的图标（Font Awesome图标），不会报错。
