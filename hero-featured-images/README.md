# 首页精选区域图片

## 📁 文件夹说明

此文件夹用于存放首页"KOKOmall精选"区域的4个橙色块图片。

## 📋 图片文件命名

请按照以下命名规则添加图片：

1. `hero-featured-1.jpg` - 第一个橙色块
2. `hero-featured-2.jpg` - 第二个橙色块
3. `hero-featured-3.jpg` - 第三个橙色块
4. `hero-featured-4.jpg` - 第四个橙色块

## 📐 图片尺寸建议

- **推荐尺寸**: 宽度与页面宽度一致，高度 94px
- **格式**: JPG 或 PNG
- **优化**: 建议压缩图片以提升加载速度

## 🔗 自动链接

图片会自动链接到首页的4个橙色块区域，无需手动修改代码。

如果图片加载失败，橙色块会保持橙色背景。
